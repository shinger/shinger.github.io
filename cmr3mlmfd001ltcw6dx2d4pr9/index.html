关于汇率的一些记录和思考

笙歌/二零二六年 六月 二十六日/约 992 字/阅读约 3 分钟

本文的知识梳理主要来自于尾河真树的《汇率原来如此有趣》

区分短中长期的外汇市场趋势

汇率是两国之间价值流向的平衡关系，分不同跨度来看其影响因素：

  • 长期：由经常账户盈余这样的结构性因素决定，跨度在2-5年。
  • 中期：由经济周期和利率趋势决定，资金倾向于流入利率高的地方，跨度在半年至2年内。
  • 短期：由市场情绪决定，可预测性低，跨度在几天至半年内。

全球日均外汇交易额约7.5万亿美元，其中实体贸易（进出口、旅游、劳务）仅占不到10%，剩下的90%都来自于利差套利、跨境债券 / 股票买卖、汇率投机、央行外汇互换、对冲基金套息等资本流动。

另外，不同的出口模式和外汇管制会对汇率造成不同的影响。比如日本的经常账户盈余主要来自于海外投资收益，不结汇，收益很少回流，加之资本管制宽松，个人可自由投资美股、债券等，大部分海外收益用于再投资。而中国的海外出口则更多来自于直接货物贸易，海外收益要求合理回流，因此大量的外汇结现导致人民币汇率更易走强，如近一年内人民币汇率持续走强。

本国货币贬值的利弊

利：

  1. 货币贬值促进对外贸易和外汇收益
  2. 本国商品的海外售价降低，利好出口
  3. 进口商品价格相对升高，提高国产商品竞争力
  4. 海外企业成本升高，促进产业回流
  5. 吸引海外投资

弊：

  1. 进口依赖型国家的成本提高
  2. 外来劳动力流失
  3. 国债海外持有者抛售（中日国债一般为内债，不像美债那样由大量海外投资者持有，故美债对汇率波动更为敏感）

国际收支账户概念辨析

国际收支恒等式：经常账户差额 + 资本账户差额 + 金融账户差额 = 0

  • 国际收支
    • 经常账户：买卖商品、服务、收入、无偿转移（实实在在的货物 / 劳务 / 钱无偿送出）
    • 金融账户：买卖股票、债券、存款、直接投资、贷款（钱用来投资、放贷、买海外资产）

日本属于经常账户顺差，但是大量资金流向海外资产，导致金融账户逆差，由于经常账户和金融账户一增一减，两者应当是平衡的，但是金融账户涉及的是投资资金，两者变化时间上不一致，因此才出现了有时升值有时贬值。比如，当经常账户盈余时，海外资本市场却出现动荡，市场情绪倾向于风险回避，对外投资就会减少，短时间内日元就会升值。

从汇率与经济周期角度看日本的持续低利率

正常情况下市场经济是周期性波动的：过热->通胀->加息->衰退->降息->复苏

然而日本的外汇市场似乎没有遵循周期规律，如果日元贬值带动了产业、资本回流，推升对外贸易，进而推动日元升值，但是日元在近十年内都在持续贬值。原因如下：

  1. 日元贬值带动的海外贸易没有传导到日本国内，大部分收益仍用于海外再投资。
  2. 日元贬值带来输入性通胀推升物价上涨，但是没有传导到服务价格，内需起不来导致内部经济无法正向循环。

相反，美国的经济周期是强需求驱动，消费能够持续为经济循环供血。
